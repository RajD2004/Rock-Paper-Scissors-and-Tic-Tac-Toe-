# C++ Mini Projects

This repo contains two small C++ console programs: a Rock–Paper–Scissors game and a Tic-Tac-Toe configuration validator.

## Files
- `main.cpp` – Rock–Paper–Scissors game :contentReference[oaicite:0]{index=0}  
- `tictactoe.cpp` – Tic-Tac-Toe board configuration checker :contentReference[oaicite:1]{index=1}  

## How to Compile & Run

### Rock–Paper–Scissors
```bash
g++ main.cpp -o rps
./rps
```
### Tic-Tac-Toe 
```bash
g++ tictactoe.cpp -o ttt
./ttt
```
